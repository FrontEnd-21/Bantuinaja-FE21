Deskirpsi Bantuan

Bantuan pendidikan jenjang S1 Merupakan program bantuan yang diadakan oleh NUS University adapun bantuan yang diberikan berupa :

  • SPP dan uang gedung gratis selama perkuliahan
  • Bantuan Uang Saku Rp 4.000.000/Semester
  • Bantuan Dana Buku Sebesar Rp 500.00

Persyaratan Bantuan

  1. KTP
  2. Buku Rapor SMA ( Sudah Legalitas )
  3. TOEFL ( Optional )
  4. Surat Keterangan Sehat Jasmani ( Terbaru )
  5. Surat Keterangan Tidak Mampu

*Point 1-4 dijadikan 1 file dan berbentuk ZIP

Tahapan Bantuan

  1. Lengkapi Dokumen

  2. Verifikasi Dokumen

  3. Pengajuan Berhasil
